Markdown格式语法(基础篇)

1.斜体、粗体

html

1.斜体:可以用标签来表示

eg:斜体

2.粗体:可以用标签来表示

eg:粗体

Markdown

1.斜体:可以用*或_引用

eg:斜体 斜体

2.粗体:可以用**或__引用

eg:粗体 粗体

2.标题 h1~h6

html

h1~h6标签来表示标题等级
eg:

Markdown

等级标题用#来表示，#越多标题越小，最多好像只有六个，再多就无效了 eg:

eg:# 一级标签

注意：补充！标题+回车+‘====’(两个等号以上)=一级标题

​ 标题+回车+‘------’(两个减号以上)=二级标题

3.超链接

html

一对a标签，教你超链接！！hahaha~~

eg:bilibili

Markdown

语法标识:[]内为文本,()为链接地址;另一种为<>直接加地址

eg:bilibili

eg:https://www.bilibili.com/

html

上标标签是,下标标签是(向上指的有没有haha~~)

Markdown

上标呢是被^围起来的；下标没有向下的，那就用~浪起来吧！

eg:这是一个小 ^ 上标 ^ 这是一个 ~ 下标 ~
