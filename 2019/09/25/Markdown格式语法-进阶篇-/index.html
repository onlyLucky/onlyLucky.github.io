5.自定义div

用不少于3个的:,冒号后面紧跟的是类名(不是所有的编译器都能够支持这种效果，不显示也很正常)

eg:

效果展示：

:::normol

hello world

:::

6.表格

html中的table，tr,th,td标签就不再多赘述了，重点是markdown的语法格式

eg:

效果展示：

th1 th2 th3
hello hello hello

注意： 其中的冒号分别代表着左对齐、居中对齐、右对齐,中间至少要用一个减号分割(一些md编译器不支持表格布局);中间减号在一些编译器中有一些权重，减号越多列宽越大
